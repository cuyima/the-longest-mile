# The Longest Mile - Content Pack
Foundry VTT module for The Longest Mile Pathfinder 2e campaign.

## Content

- **Dervish Archetype** - Psi-based archetype with automated focus point management and strike effects
- **Deities** - 35 custom deities for the campaign setting
- **Visual Changes** - CSS overrides for character sheets, Simple Calendar, and combat tracker to match campaign aesthetics
- **House Rule** - Keeley's Hero Points variant that only applies during combat (adds +10 to hero point rerolls of 10 or lower)

## Release

1. Update version in `static/module.json`
2. Tag the commit: `git tag v1.x.x`
3. Push tags: `git push --tags`
4. GitHub releases are created automatically
